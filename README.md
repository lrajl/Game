🎮 Tic Tac Toe Game by Rishi Raj
A simple, fast, and interactive Tic Tac Toe game developed by Rishi Raj using pure JavaScript. This project is perfect for beginners who want to understand game logic and DOM manipulation.
🚀 Features
👥 Two-player mode (Play with your friend)
⚡ Smooth and fast gameplay
🎯 Instant win/draw detection
🔄 Restart button to play again
📱 Responsive design (Mobile + Desktop)
